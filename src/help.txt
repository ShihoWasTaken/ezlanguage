NAME
	EZL
	
SYNOPSIS
	EZL [options] files..
	for the options list, see the options section below.
	
DESCRIPTION
	This is the EZ language compiler, a C based language for beginners.
	
EXAMPLES
	Some xamples of common usage :
	EZL example1.ez example2.ez -o example.exe
	
OPTIONS
	--directinput			: Enable direct input for EZ language
	-h, --help			: Displays this information
	--noexec			: Do not launch the executable
	-o <file>, --output=<file>	: Name the executable <file>
	-O1, --optimisation=1		: Reduces the execution time, first level of optimization
	-O2, --optimisation=2		: Same as O1 lvl2
	-O3, --optimisation=3		: Same as O2 lvl3
	-v, --verbose			: Displays all the compilation steps in the command prompt
	-w, --warning			: Displays all the warning messages
